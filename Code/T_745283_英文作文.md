### 1. 英文作文题逻辑拆解

* **步骤 A：处理首字母**
  * 根据操作 2，首字母如果为小写，必须改为大写。
* **步骤 B：处理中间字符（遍历原字符串）**
  * 遇到 **大写字母** ：如果它前面不是句点 `.`，就在它前面加一个 `.`。
  * 遇到 **其他字符** （小写字母或原本就有的句点）：直接保留。
* **步骤 C：处理末尾句点**
  * 根据操作 1，如果处理后的字符串最后一个字符不是 `.`，必须在末尾补一个 `.`。
* **注意点** ：题目提到“不存在两个相邻的句点”，这简化了逻辑，意味着我们不需要考虑连着两个 `..` 的去重问题。4

---

代码实例：

#include `<iostream>`
#include `<string>`
#include `<cctype>` // 包含 toupper 和 isupper 函数

using namespace std;

int main() {
    string s;
    cin >> s;

    string res = "";

    for (int i = 0; i < s.length(); ++i) {
        // 处理首字母大写
        if (i == 0) {
            res += toupper(s[i]);
        }
        // 处理非首字符的大写字母
        else if (isupper(s[i])) {
            // 如果前一个字符不是句点，先补句点
            if (res.back() != '.') {
                res += '.';
            }
            res += s[i];
        }
        // 处理句点后的首字母大写 (操作2)
        else if (i > 0 && s[i-1] == '.') {
            res += toupper(s[i]);
        }
        // 普通小写字母或已有句点直接添加
        else {
            res += s[i];
        }
    }

    // 操作1最后一条：如果结尾不是句点，补上
    if (res.back() != '.') {
        res += '.';
    }

    cout << res << endl;

    return 0;
}
